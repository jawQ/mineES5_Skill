# Array

## 方法：
#### 排序：
1. 升序: arr.sort((a,b)=> {a-b})  会改变源数组

#### 倒序：
1. reverse()

#### 去重：
1. 结合splice

#### 合并:
1. concat


#### 删除:
1. [pop()][pop]  ---  删除并返回数组的最后一个元素。
2. [shift()][shift]   ---  删除并返回数组的第一个元素。
3. [splice()][splice]  ---  splice() 方法向/从数组中添加/删除项目，然后返回被删除的项目。
4. [delete arr[i]][delete]  ---  把数组当成对象，索引值作为属性值处理。
